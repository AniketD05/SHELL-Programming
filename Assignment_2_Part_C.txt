Question 4: Write a shell script that performs addition of two numbers (e.g., 5 and 3) and prints the
result.

Solution--->

Step 1 : vi A2_CQ4.sh

	#!/bin/bash

	echo "Enter Number 1: "
	read num1
	echo "Enter Number 1: "
	read num2

	echo "Entered numbers are $num1 $num2"

	sum=$(( num1+num2 ))

	echo "Sum is $sum"

Step 2 : chmod +x A2_CQ4.sh
Step 3 : ./A2_CQ4.sh

output --> Enter Number 1: 
		10
		Enter Number 1: 
		20
		Entered numbers are 10 20
		Sum is 30

Question 5: Write a shell script that takes a number as input and prints "Even" if it is even, otherwise prints "Odd".

Solution--->

Step 1 : vi A2_CQ5.sh

	#!/bin/bash

	echo "Enter Number:"
	read num

	if(( num%2==0  ))
	then
        echo "Even"
	else
        echo "Odd"
	fi

Step 2 : chmod +x A2_CQ5.sh
Step 3 : ./A2_CQ5.sh

output --> Enter Number:
	   5
	   Odd
	   4
	   Even
